Bonjour et bienvenue dans le projet réseau.

1) Accès base de données :

Lancer le main vous permettra d'accéder à la base de données depuis une interface utilisateur textuelle. Vous pourrez vous connecter à partir des identifiants et mots de passe déjà présents dans la base de données. L'interface vous guidera ensuite vers les différentes fonctionnalités pouvant être explorer. Dans les grandes lignes vous pourrez visualiser toutes les informations sur les autres utilisateurs ainsi que leur postes, et vous pourrez aussi ajouter vos postes, les modifier, et les supprimer. 

Fonctionnalités encore en développement :
Une fonctionnalité pour liker les postes existe, cependant elle n'est pas finalisée (un utilisateur peut liker autant de fois qu'il le souhaite).
Les réponses aux postes s'affichent sur les murs mais l'auteur indiqué reste celui de l'auteur du mur.
Vous pouvez supprimer et modifier des postes mais seulement à conditions d'avoir leur identifiant de poste.
Pour l'instant un utilisateur bloqué ou un non followers peut quand même visualiser les postes qu'il souhaite.
On ne peut ni rajouter d'utilisateur ni modifier leur informations.
On peut ajouter un poste même sans avoir de mur.
On ne peut pas disliker, bloquer, debloquer, follow ou unfollow quelqu'un.

2) Accès logiciel :

Si vous décidez de lancer uniquement la partie logicielle, vous aurez accès à un peu près toutes les fonctionnalités qui ne sont pas encore disponibles pour l'accès base de données. Vous pourrez aussi ouvrir des images.
Une partie test de toutes ces fonctionnalités est présente dans le main sous forme de commentaires. Pour voir son fonctionnement enlever simplement les */ de part et d'autre du bloc de code.
